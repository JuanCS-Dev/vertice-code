INSTALLATION
$ pip install vertice-code

Python 3.10+ required. Works on macOS, Linux, and Windows (WSL).

Core Commands

Interact with agents using natural language. Simple, intuitive, powerful.

vertice chat

Start an interactive conversation with agents.

$ vertice chat "explain this code"

vertice run

Execute a task with agent orchestration.

$ vertice run --task "refactor tests"

vertice --mode omni

Launch the Terminal UI for real-time monitoring.

$ vertice --mode omni

vertice config

Configure API keys and preferences.

$ vertice config set ANTHROPIC_API_KEY
